Talk to the Controller Prototype ver 1.0

CONTROLS:
- Use the arrow keys to walk around
- Press the [Z] button to interact with people/objects
- Use the [Mouse Buttons] to select options when appropriate

Main Level:
- Interact with the capsule to start the battle

Battle Scene:
- You have three choices during battle: Attack, Defend or Grab
- Your opponent will select an option the same time as you do.
- You win if you reduce your opponent's health to 0.
